    • DIBUJO DE AVES

    Indice

    • Materiales
    • Paso 1: Boceto
    • Paso 2: Capas base
    • Paso 3: Detalles / textura

    Materiales Necesarios:

    • Papel de alto gramaje: para que resista muchas capas de carboncillo.
    • Lapiz de carboncillo: para realizar detalles (preferentemente de dureza media / dura).
    • Carboncillo en barra: para realizar las capas base.
    • Lapiz pastel blanco: será utilizado para remarcar partes donde hay mayor iluminación y parte del pelaje.
    • Borrador: cualquier goma o borrador que tengas disponible.
    • Servilleta: esto sera usado para difuminar el carboncillo.

    ETAPA 1: Boceto

    En esta etapa inicial se busca lograr la estructura base del dibujo y las proporciones correctas, para esto como primera acción se utilizan figuras geometricas simples como se muestra en la imagen de referencia, esto brinda una buena base para ubicar las otras partes de la figura. En este paso se realizan trazos simples y suaves, los detalles serán realizados mas adelante.

    ETAPA 2: Capas base

    En este paso lo que se busca lograr son las capas base, es decir los tonos principales de la imagen. Utilizando los carboncillos en barra y la servilleta para difuminar. para esto hay que prestar atención a los niveles de luz y sombra de la figura, para lograr las tonalidades correctas. una practica util en este caso es hacer una paleta de colores en una pedazo de hoja aperte, como se muestra en la imagen de referencia. Importante: en caso de aplicar varias capas sobre una misma parte se debe comenzar por los tonos y colores MAS OSCUROS.

    ETAPA 3: Detalles y textura

    En esta ultima etapa nos vamos a centrar en dar textura y los detalles, aqui vamos a utilizar los lapices de carboncillo ya que permiten dar trazos mas precisos. Para esto es muy importante analizar la composición del pelaje, particularmente caracteriscas como las siguientes: largo, ondulación y grosor. Como ayuda puedes realizar en una hoja aparte una muestra de una parte del pelaje, para tener una idea mas clara de su composición. Esta etapa siempre es la que mas tiempo demanda, es necesario tomarse esta parte con mucha paciencia.
