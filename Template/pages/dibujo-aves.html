    Lista de materiales

    1. Papel:hojas de gramaje alto o medio para que pueda resistir varias capas de carboncillo.
    2. Carboncillo en barra: para realizar fondos y capas bases.
    3. Lapiz de carboncillo: para realizar detalles.
    4. Borrador: se utilizara para crear brillos y texturas (puede ser uno comun o moldeable).
    5. Servilleta: para difuminar el carboncillo con mayor facilidad. (tambien es recomendable usar un difuminador para detalles mas pequeños)

    PASO 1: boceto

    En esta etapa se busca lograr una estructura base de la figura que estamos dibujando, para ello primero se realizan simples figuras geometricas para comenzar a definir las proporciones. en esta etapa NO uno no se debe enfocar en los detalles, sino en definir una buena estructura inicial para lograr las proporciones correctas.

    PASO 1: capas base & tonalidades

    En este paso se comienza a trabajar con las capas bases, estas definen las tonalidades de la figura, para ello es recomendable usar los carboncillos en barra ya que tienen mayor capacidad de difuminación, por cierto para difuminar el material puedes hacerlo directamente con la mano pero por una cuestion de prolijidad lo recomendable es hacerlo con una servilleta. Generalmente se comienza aplicando las capas mas oscuras y gradualmente se prosigue con los tonos mas claros. Para trabajar de una manera mas efectiva te sugiero que primero analices las fuentes de luz y sombra de la figura, esto te va a dar una mayor orientacion a la hora de definir los contrastes.

    PASO 3: Textura y detalles

    Finalmente esta es la ultima etapa y generalmente la que mayor tiempo lleva. Al igual que el paso anterior se comienza con las areas mas oscuras y gradualmente se va avanzando a las areas mas claras. Aquí es escencial analizar la composicion del pelaje y los "patrones" que se forman en el mismo. Puedes tomar algun pedazo de hoja aparte e ir haciendo pequeñas pruebas y estudios del pelaje antes de trabajar en la obra principal.
