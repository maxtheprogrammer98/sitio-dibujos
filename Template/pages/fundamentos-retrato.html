    Lista de materiales

    1. Papel:hojas de gramaje alto o medio para que pueda resistir varias capas de carboncillo.
    2. Carboncillo en barra: para realizar fondos y capas bases.
    3. Lapiz de carboncillo: para realizar detalles.
    4. Borrador: se utilizara para crear brillos y texturas (puede ser uno comun o moldeable).
    5. Servilleta: para difuminar el carboncillo con mayor facilidad. (tambien es recomendable usar un difuminador para detalles mas pequeños)

    PASO 1: boceto

    En esta etapa lo que se busca lograr es la estructura base de la figura, para ello lo primero que se hace es trazar figuras simples como referencia tal como se puede observar en la imagen superior. El circulo representa la mitad superior del craneo y la cuadricula dividida en 3 partes representa la ubicacion de los rasgos del rostro. Lo importante en este paso es definir las proporciones correctas, esto es como los cimientos de nuestra obra. Es decir es una etapa fundamental en la que no deben haber errores, no dudes en tomarte un tiempo extra para verificar que todas las proporciones sean correctas.

    PASO 2: Capas bases & rasgos faciales

    En esta etapa se comienza a aplicar el carboncillo, yo generalmente prefiero empezar definiendo el rostro y dejo la parte del cabello y el fondo para después. lo importante en este paso es lograr las tonalidades correctas, para esto puedes ayudarte haciendo una paleta de colores. Una vez que hayas carboncillo en una area puedes difuminarlo con la servilleta o el difuminador. Como puedes ver en la imange de referencia todavia no se logra un gran nivel de detalle, nos enfocaremos en ese aspecto en la siguiente etapa. (ten en cuenta que siempre se recomienda empezar por las capas mas oscuras, luego sobre las mismas puedes agregar brillos o tonos mas claros).

    PASO 3: Detalles y textura

    Finalmente comenzamos a trabajar en los detalles y la textura. En esta etapa vamos a emplezar casi exclusivamente el lapiz de carboncillo ya que nos permite crear trazos mas finos y precisos. Este es un proceso bastante repetitivo donde basicamente se agrega capas tras capa de detalles. En el caso del cabello, una vez que se haya hecho el fondo se comienza a trazar los mechones mas claros con el borrador.
